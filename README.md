[ NEDİR? ]
Bu program, Rusça oyunlarda veya uygulamalarda ekranda gördüğünüz
yazıları seçip anlık olarak Türkçeye çevirmenizi sağlar.
Kopyalanamayan oyun içi metinleri (görevler, eşya isimleri vb.)
okumak için tasarlanmıştır.

-------------------------------------------------------------

[ NASIL KULLANILIR? ]

1. "cevirici.exe" dosyasını yönetici olarak çalıştırın.
2. Program açıldığında alt kısımdaki durum çubuğunda:
   "Durum: Hazır!" yazısını bekleyin. (İlk açılış 10-15 sn sürebilir).
3. Oyuna girin.
4. Çevirmek istediğiniz yazının olduğu yeri klavyenizden:
   
   >>>  WINDOWS TUŞU + SHIFT + S  <<<
   
   tuşlarına aynı anda basarak kare içine alın.

5. Siz seçimi bırakır bırakmaz program otomatik olarak o bölgeyi
   algılar, Rusça yazıyı okur ve Türkçeye çevirir.
6. Sonuçlar program penceresinde anında belirir.

-------------------------------------------------------------

[ SIK SORULAN SORULAR & İPUÇLARI ]

* Program Açılmıyor / Yavaş Açılıyor:
  Programın içinde yapay zeka motoru (EasyOCR) olduğu için ilk açılışta
  dosyaları yüklemesi 10-20 saniye sürebilir. Lütfen sabırlı olun.

* Çeviri Yapmıyor:
  Windows panosunu (Clipboard) kullanır. "Win+Shift+S" yaptıktan sonra
  resmin panoya kopyalandığından emin olun (Bildirim gelmeli).

* Virüs Uyarısı Veriyor:
  Program imzalanmamış bir .exe olduğu için bazı antivirüsler
  şüpheli görebilir. Tamamen güvenlidir, istisnaya ekleyebilirsiniz.
